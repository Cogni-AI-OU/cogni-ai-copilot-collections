# Plugins

Portable bundles that package agents, skills, and configuration together for easy installation.

## Available Plugins

- **[cogni-ai-coder](cogni-ai-coder/)** — Autonomous coding agent with critical-thinking skill (1 agent, 1 skill)
- **[cogni-ai-programmer](cogni-ai-programmer/)** — Autonomous programmer agent with Python skill (1 agent, 1 skill)

## Installation

```bash
copilot plugin install https://github.com/Cogni-AI-OU/cogni-ai-copilot-collections/tree/main/plugins/<plugin-name>
```

Each plugin includes a `README.md` with detailed installation and usage instructions.
